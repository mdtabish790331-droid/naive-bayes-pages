# 📊 Naive Bayes Educational Dashboard

An interactive web-based learning system designed to help students understand and visualize the **Naive Bayes algorithm** step-by-step.


## 🚀 Features

- 📂 Upload CSV dataset
- 🔍 Data preview (head, shape, columns)
- 🧹 Preprocessing explanation
- 📊 Exploratory Data Analysis (EDA)
  - Distribution plots
  - Class visualization
- 🧠 Naive Bayes Algorithm Explanation
  - Prior Probability
  - Likelihood
  - Posterior Probability
- ⚙️ Model Training (basic simulation)
- 🔮 Prediction Module (interactive)
- 📈 Evaluation Metrics
  - Accuracy
  - Precision
  - Recall
  - F1 Score
- 🎨 Clean UI with animations
- 📌 Sidebar progress tracking


## 🧠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- PapaParse
- MathJax
